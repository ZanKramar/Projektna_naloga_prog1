A lot of content,The game is not only astounding technically speaking (the graphics push the PSX to its limits) but it is also a joy to play. A lot of content, eight ships, eight tracks plus four prototype tracks (and if you get the Special Edition, there are two more prototypes and eight more tracks from the two previous games!) spanning through four speed classes, 32 challenges and four tournaments. Considering that Rapier Class and Phantom Class start to get really serious in terms of difficulty, there is a lot to do.
    The game is also very customizable (you can remove weapons, trim the number on the grid, and other things via the secret cheat codes) and it offers a very good and immersive cockpit view, which is a huge plus in my opinion.
    The soundtrack is very spot-on with a lot of good tunes as always.

    Honestly, Wipeout 3 is as good as it can be on the PSX if you love the genre, and it is a shame that it didn’t get the attention and the success it deserved.    … Expand
